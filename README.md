# Stickers App

Десктопное приложение для создания стикеров-заметок на рабочем столе.

## Возможности
- Создание неограниченного количества заметок
- Изменение цвета и шрифта каждой заметки
- Режим "Поверх всех окон"
- Прикрепление к экрану (не выходит за границы монитора)
- Автосохранение заметок
- Работа из системного трея
- Поддержка нескольких мониторов
- Плавные анимации

## Требования
- Windows 7/8/10/11
- Python 3.8+ (для запуска из исходников)

## Установка
### Способ 1: Готовый EXE файл
1. Скачайте `Stickers.exe` из раздела [Releases](https://github.com/DaniiL-Buzakov/Stikers/releases)
2. Запустите файл

### Способ 2: Запуск из исходников
```bash
# Установите зависимости
pip install pyqt5

# Запустите приложение
python main.py
